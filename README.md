# GatherApp

A mobile social platform for campus communities built with Kotlin and Jetpack Compose.

## Overview

GatherApp enables students to share posts, report lost items, list marketplace goods, post safety alerts, and engage through likes and comments with real-time notifications.

## Features

- User authentication with Firebase
- Create posts across multiple categories (Marketplace, Lost & Found, Safety)
- Like and comment on posts
- Real-time notifications with toast alerts and badge counters
- View who liked each post
- Filter posts by category
- Upload images with posts

## Technology Stack

**Frontend**
- Kotlin
- Jetpack Compose
- Material Design 3
- Navigation Compose
- Coil (image loading)

**Backend**
- Firebase Authentication
- Cloud Firestore (database)
- Supabase Storage (images)

**Architecture**
- MVVM Pattern
- Kotlin Coroutines
- Singleton Pattern

## Usage

1. Sign up or login
2. Browse posts or filter by category
3. Create posts with the floating action button
4. Like and comment on posts
5. Receive real-time notifications
